Install Debian GNU/Linux Jessie 8.7 Dengan NetInstall

Thu 03 January 2013 in GNU/Linux,

Share Tweet Share

Cara install debian GNU/Linux netinstall, How to install Debian Netinstall

Pada postingan kali ini akan membahas cara installasi debian menggunakan NetInstaller cd / iso.

Sesuai nanamnya NetInstall, maka installer ini memerlukan internet untuk proses installasinya, dimana dia hanya menyertakan paket dasar minimal agar system dapat menjalankan atau booting live environment untuk proses instalasi saja, sama seperti installer arch, gentoo stage tar ball ataupun mfsBSD.

Download lah netinst ISO dari mirror atau repo utama Debian. Sebagai contoh disini saya mendownloadnya dari repo mirror dan memilih untuk arsitektur 64bit
debian-8.7.1-amd64.iso
jika ingin menginstall versi 32 bit maka pilih yang i386 seperti berikut
debian-8.7.1-i386.iso

Setelah selesai terunduh atau download, maka buat installernya. Jika menggunakan windows dan ingin menginstall menggunakan cd maka bisa tinggal burning saja installer tersebut menggunakan tools cd/dvd burner favorite kalian, jika ingin membuat installer USB installer maka bisa menggunakan rufus. Saya pribadi selalu menggunakan rufus ketika membuat usb installer dari windows system, alternative bisa menggunakan:

Jika menggunakan system linux, OSx/Mac ataupun unix like lainnya maka bisa menggunakan dd, dengan menjalankan perintah berikut:

# dd if=debian.iso of=/dev/sdX

catatan:

  • if adalah input file atau image atau iso file yang akan di buat menjadi installer
  • of adalah output file atau disk tujuan yang akan kita jadikan installer.

warning: Cek dengan pasti dulu disk tujuan, karna salah disk tujuan ketika menggunakan dd command akan membuat data hilang
